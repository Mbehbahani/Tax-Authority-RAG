# Sample Requests Before Running

Use these files as prepared inputs before any local PoC or test runner exists. They help Claude Code understand expected API inputs, user roles, and expected behavior.

## Files

- [`users.json`](users.json): synthetic users and access scopes.
- [`queries.json`](queries.json): sample queries for exact retrieval, semantic retrieval, RBAC denial, zero-hallucination, cache, and performance smoke tests.
- [`expected_behaviors.json`](expected_behaviors.json): expected retrieval/generation behavior per query.

These files are synthetic and must not be treated as real tax advice.

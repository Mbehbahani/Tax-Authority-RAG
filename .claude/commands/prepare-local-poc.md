# Command: Prepare Minimal Local PoC Later

Prepare a minimal local proof of concept only when explicitly requested.

## Scope

- FastAPI endpoint.
- Local sample legal documents.
- Local retrieval backend or mock OpenSearch.
- LangGraph CRAG flow.
- RBAC test users.
- Optional Redis.
- Demonstrate chunking, retrieval, RBAC filtering, CRAG grading, abstention, and citation formatting.

## Boundaries

- No real AWS deployment.
- No real credentials.
- No production infrastructure.
- Keep the PoC small and focused on validating architecture claims.

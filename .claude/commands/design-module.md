# Command: Design One RAG Assessment Module

Use this command to design a single module of the Enterprise Tax Authority RAG assessment.

## Steps

1. Restate the module objective and scope.
2. Identify risks, failure modes, and assessment constraints.
3. Propose architecture choices with rationale.
4. Specify exact configuration parameters and defaults.
5. Provide pseudo-code or structured flow.
6. Define unit, integration, security, evaluation, and performance tests.
7. Explain tradeoffs and alternatives.

## Required Emphasis

- Exact legal citations: document, article, paragraph.
- RBAC-before-retrieval.
- Zero-hallucination behavior with abstention.
- Scale to 500k documents and 20M+ chunks.
- TTFT, p95/p99 latency, OOM risk, and memory pressure.
